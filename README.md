# pro-zhajinhua-guide

本项目用于归档和发布多个独立的 HTML 页面，这些页面内容均不针对任何特定域名或网站。仓库以静态文件形式存放，便于访问和查阅。

## 项目简介

这是一个以 HTML 页面归档为主要目的的仓库。页面内容涵盖多个主题，均以独立、自包含的 HTML 文件形式存在。这些页面不依赖外部动态服务，可直接在浏览器中打开查看。

## 目录说明

```
pro-zhajinhua-guide/
├── index.html          # 入口页面（如存在）
├── pages/              # 存放各独立 HTML 页面
│   ├── example-1.html
│   ├── example-2.html
│   └── ...
├── assets/             # 公共资源（CSS、图片等）
│   ├── style.css
│   └── images/
└── README.md           # 本文件
```

- **index.html**：可选，作为仓库的起始页面。
- **pages/**：核心目录，存放所有独立的 HTML 页面文件。
- **assets/**：存放页面所需的公共资源，如样式文件、图片等。

## 页面归档说明

所有页面均为完整的 HTML 文档，包含独立的样式与结构。每个页面不依赖外部 API 或数据库，仅通过本地或静态托管即可正常展示。页面内容可能涉及不同主题，但均不指向任何具体域名、网站或商业服务。

## 维护说明

- 本仓库主要面向个人归档与分享需求。
- 如需添加新页面，建议在 `pages/` 目录下创建独立 HTML 文件，并确保文件内无外部依赖。
- 资源文件建议统一放入 `assets/` 目录，以保持项目整洁。
- 不鼓励在页面中加入外部链接、跟踪代码或营销内容。
- 欢迎通过 Issue 提出建议或问题，但请注意本仓库不提供针对特定网站的技术支持。
